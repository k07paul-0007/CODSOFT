# AI Internship Projects

This repository contains three Artificial Intelligence internship projects developed using Python. Each project demonstrates a different AI concept, ranging from rule-based systems to game AI and recommendation systems.

---

## 📁 Projects Included

### 1️⃣ Rule-Based Chatbot

**Description**
A simple chatbot that responds to user queries using predefined rules and conditional statements.

**Features**
- Greeting responses
- AI and Python information
- Date and Time
- Help command
- Exit command

**Technology Used**
- Python
- if-elif-else statements
- datetime module

**Run**

```bash
python chatbot.py
```

---

### 2️⃣ Tic-Tac-Toe AI

**Description**
A console-based Tic-Tac-Toe game where the user plays against an AI. The AI uses the Minimax algorithm to choose the best possible move.

**Features**
- Human vs AI gameplay
- Minimax Algorithm
- Win, Loss, and Draw detection
- Interactive console interface

**Technology Used**
- Python
- Minimax Algorithm

**Run**

```bash
python tic_tac_toe.py
```

---
